# MPI Lens — Poverty Prediction Dashboard

## Objective
Predict Multidimensional Poverty Index (MPI) using socio-economic indicators.

## Models Used
1. Multiple Linear Regression (Baseline)
2. Random Forest Regressor (Advanced)
3. Gradient Boosting / XGBoost (Optional)

## Features
- Model comparison
- Feature importance
- State ranking
- What-if simulation dashboard

## How to Run
streamlit run app.py
